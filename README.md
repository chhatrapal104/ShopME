\## 🚀 Overview



ShopME is a modern e-commerce frontend application built using Next.js and Tailwind CSS.  

The project focuses on scalable frontend architecture, reusable UI components, and real-world debugging experience.



\---



\## 🧠 Key Highlights



\- Built using \*\*Next.js App Router (latest architecture)\*\*

\- Implemented \*\*reusable UI component system\*\*

\- Managed global state using \*\*Zustand\*\*

\- Used \*\*Radix UI primitives\*\* for accessible components

\- Fixed real-world issues:

&#x20; - Dependency conflicts

&#x20; - Build errors

&#x20; - Missing modules

&#x20; - Deployment issues
